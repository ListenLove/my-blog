# MyBlog - 简易静态博客系统

MyBlog 是一个简易的静态博客系统，类似于 Hugo，专为命令行使用而设计。它借助 GitHub 对 Markdown 文档的完美支持，让 Markdown 文档成为你的界面。

## 特性

- 🚀 **简单易用**: 基于命令行的简洁界面
- 📝 **Markdown 支持**: 完美支持 Markdown 格式
- 🏷️ **标签和分类**: 支持文章标签和分类管理
- 🎨 **交互式命令**: 美观的交互式命令行界面
- 🌈 **彩色输出**: 支持彩色终端输出
- 📁 **自动组织**: 自动创建和组织文件结构

## 快速开始

### 构建项目

```bash
go build -o myblog .
```
